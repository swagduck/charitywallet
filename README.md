# Charity Wallet

A decentralized application (DApp) for managing and donating to charitable causes using blockchain technology.

## 🌟 Features

- **Secure Donations**: Make secure donations using cryptocurrency
- **Transparent Transactions**: All transactions are recorded on the blockchain
- **Multiple Wallets**: Support for various wallet connections
- **Charity Discovery**: Browse and discover different charitable organizations
- **Donation Tracking**: Keep track of your donation history

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- A Web3 wallet (like MetaMask)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/charity-wallet.git
   cd charity-wallet
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add the following:
   ```
   REACT_APP_INFURA_ID=your_infura_id
   REACT_APP_NETWORK=mainnet
   ```

4. Start the development server:
   ```bash
   npm start
   # or
   yarn start
   ```

## 🛠 Built With

- [React](https://reactjs.org/) - Frontend library
- [Ethers.js](https://docs.ethers.org/) - Ethereum wallet implementation
- [Web3Modal](https://github.com/Web3Modal/web3modal) - For wallet connection
- [TailwindCSS](https://tailwindcss.com/) - For styling

## 📄 Smart Contracts

The project uses smart contracts deployed on the Ethereum blockchain. The main contracts include:

- `CharityFactory.sol` - For creating and managing charities
- `Donation.sol` - For handling donations
